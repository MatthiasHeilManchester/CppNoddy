\doxysubsubsection{Detailed Description}
A simple arc-\/length continuation solving the equation \[ x^2 + p^2 = 2\,, \] where $p$ is a parameter. The solution is obtained by Newton iteration combined with (pseudo) arclength continuation with the parameter $p$. For a (slightly) more serious example see the Falkner\+Skan\+\_\+arc.\+cpp example.
