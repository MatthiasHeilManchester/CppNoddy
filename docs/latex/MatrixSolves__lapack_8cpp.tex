\doxysubsubsection{Detailed Description}
\doxyref{Example}{p.}{namespaceExample} of the simple linear solvers implemented for dense, banded and sparse matrix objects. To begin with a simple $ 2 \times 2 $ matrix problem is solved. Then a penta-\/diagonal problem is solved using the dense, banded and sparse containers. The native linear Gaussian elimination solvers are used unless the P\+E\+T\+S\+C\+\_\+\+D/Z compiler options are used, in which case the linear solver phase calls the P\+E\+T\+Sc library as appropriate.
