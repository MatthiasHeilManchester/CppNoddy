\doxysubsubsection{Detailed Description}
Solving a one-\/dimensional \char`\"{}\+Bi-\/harmonic\char`\"{} eigenvalue problem (E\+VP) \[ \left ( \frac{\mbox{d}^2}{\mbox{d}x^2} - \lambda \right )^2 f(x) = 0\,, \quad \mbox{where} \quad f(0)=f(1)=f\textnormal{\textquotesingle}(0)=f\textnormal{\textquotesingle}(1)=0\,,\] via Runge-\/\+Kutta and (vector) Newton iteration. The problem is nonlinear since $\lambda$ is unknown.

Locally solving for eigenvalues using I\+VP methods is generally speaking not a great way of doing things; this is just a \doxyref{Example}{p.}{namespaceExample} rather than any recommendation! See the other E\+VP examples for better methods.
