\doxysubsubsection{Detailed Description}
Solving the Blasius equation \[ f\textnormal{\textquotesingle}\textnormal{\textquotesingle}\textnormal{\textquotesingle}(y) + f(y) f\textnormal{\textquotesingle}\textnormal{\textquotesingle}(y) = 0\,, \] with $ f(0)=f\textnormal{\textquotesingle}(0)=0 $ and $ f\textnormal{\textquotesingle}(\infty) = 1 $ in the domain $ y \in [0,\infty ] $ by applying the O\+D\+E\+\_\+\+B\+VP class. The class constructs and solves the global matrix problem using 2nd-\/order finite differences over a finite domain of specified size.
