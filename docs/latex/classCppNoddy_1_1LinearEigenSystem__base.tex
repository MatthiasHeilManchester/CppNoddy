\doxysubsubsection{Detailed Description}
A linear Nth-\/order generalised eigensystem base class. 

Here we can construct a linear eigenproblem in the form \[ A_{NxN} \,{\underline x}_i = \lambda_i\, B_{NxN}\, {\underline x}_i \] for matrices $ A $ and $ B $. The eigenvalues and eigenvectors can be tagged and retrieved as required.
