class Cpp\+Noddy\+::\+Equation\+\_\+1matrix$<$ \+\_\+\+Type, \+\_\+\+Xtype $>$}

An equation object base class used in the I\+B\+VP classes (and others). 

An equation object is essentially a (\textquotesingle{}square\textquotesingle{}) residual object with an independent variable data member and access methods. By \textquotesingle{}square\textquotesingle{} we mean that it defines N residuals and N state variables. The equation is defined using an NxN matrix that multiplies the derivative of unknowns and a residual R\+HS.
