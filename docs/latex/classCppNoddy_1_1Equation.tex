An equation object base class used in the B\+V\+P/\+I\+VP classes. 

An equation object is essentially a (\textquotesingle{}square\textquotesingle{}) residual object with an independent variable data member and access methods. By \textquotesingle{}square\textquotesingle{} we mean that it defines N residuals and N state variables.
