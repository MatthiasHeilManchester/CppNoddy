\doxysubsubsection{Detailed Description}
Solving the Karman rotating-\/disk equations for the flow above an infinite rotating disk by applying a user-\/provided Jacobian \[ U\textnormal{\textquotesingle}\textnormal{\textquotesingle}(y) = U^2(y) + V(y)U\textnormal{\textquotesingle}(y) - W^2(y) \] \[ W\textnormal{\textquotesingle}\textnormal{\textquotesingle}(y) = 2U(y)W(y) + V(y)W\textnormal{\textquotesingle}(y) \] \[ 2U(y) + V\textnormal{\textquotesingle}(y) = 0 \] with boundary conditions $ U(0)=V(0)=0 $, $ W(0)=1 $ and $ U(\infty ) \to 0 $, $ W(\infty ) \to 0 $. The class constructs and solves the global matrix problem using 2nd-\/order finite differences.
