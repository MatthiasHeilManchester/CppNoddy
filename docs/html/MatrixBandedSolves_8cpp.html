Detailed Description

Example of a simple "banded" inear solver using matrix problem is solved. Then a penta-diagonal problem is solved using the dense, banded and sparse containers. The native linear Gaussian elimination solvers are used unless the PETSC_D/Z compiler options are used, in which case the linear solver phase calls the PETSc library as appropriate.
