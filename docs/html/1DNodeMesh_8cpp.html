Detailed Description

A simple check of the OneD_Node_Mesh container that stores nodal data over a given mesh. This checks that mesh data is interpolated and integrated correctly. We write to a uniform mesh, then check the integration routines converge appropriately. We then re-interpolate the data onto a non-uniform mesh and check the integral again.
